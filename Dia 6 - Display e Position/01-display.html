display

É a propriedade mais importante do CSS para controle de layout

Por padrão boa parte dos elemento HTML tem os valores de block e inline e a propriedade display é utilizado para mudar esses comportamentos.

Elementos nível de bloco

Um elemento nível de bloco sempre começa em uma nova linha e ocupa toda a largura disponível.

Exemplos de elementos com nível de bloco

  • div
  • h1-h6
  • p
  • form
  • header
  • footer
  • section

O elemento div é um elemento nível de bloco

Elementos em linha

É um elemento embutido que NÃO começa em uma nova linha e ocupa apenas a largura necessária.

Exemplos de elementos inline:

  • span
  • a
  • img

Este é um elemento span embutido dentro de um parágrafo.

Alguns valores da propriedade display

block

Exibe um elemento como um elemento de bloco

A proriedade de display com um valor de block resulta em uma quebra de linha entre cada elemento span.
