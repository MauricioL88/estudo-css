As propriedades para o flex container são:

  • flex-direction
  • flex-wrap
  • flex-flow
  • justify-content
  • align-items
  • align-content

Propriedade Flex-Direction:

Especifica a direção de exibição dos flex items no flex container.

Pode conter um dos valores:

  • row
  • column
  • row-reverse
  • column-reverse

Flex-Wrap

Propriedade que especifica se os flex items devem ser quebrados ou não, se não houver espaço suficiente para eles em uma flex-line.

Pode conter um dos valores:

  • nowrap
  • wrap
  • wrap-reverse

Exemplo

nowrap especifica que os flex items não quebrarão (valor padrão).
