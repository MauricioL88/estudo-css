Flexbox

É a abreviatura de um módulo de caixa flexível

É um método para arrumar os itens em linhas (rows) ou colunas (columns)

Torna mais fácil desenhar uma estrutra com layout responsível e flexível, sem a utilização de float ou positioning

Diferença entre Flexbox e Grid

  • O Flexbox deve ser usado para layouts unidimensionais, com linhas e colunas
  • O Grid deve ser usado para layouts bidimensionais, com linhas e colunas

Módulo CSS Layout de caixa fléxível

Antes exitiam 4 modos de layout:

  • Block: para <section> de uma webpage.
  • Inline: para texto.
  • Table: para tabela de dados bidimensional.
  • Positioned: para uma posição explícita de um elemento.

Componentes Flexbox

Um flexbox sempre consiste em:

  • Flex Container - Elemento "Pai" (Container) <div>
  • Flex Itens - Itens dentro do container <div>
